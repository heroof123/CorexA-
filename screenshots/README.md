# Screenshots & Media

## How to Add Screenshots

1. Take screenshots of CorexAI IDE
2. Save them in this folder with descriptive names:
   - `main-interface.png` - Main IDE interface
   - `ai-chat.png` - AI chat panel
   - `code-editor.png` - Code editor with syntax highlighting
   - `git-panel.png` - Git integration
   - `settings.png` - Settings panel

3. Update the main README.md with image links:
   ```markdown
   ![Main Interface](screenshots/main-interface.png)
   ```
